Here's a quick summary of Day 3:

**Boolean**
- Only two values: `True` or `False` (first letter must be capitalized)

**Assignment Operators**
- `=` stores a value in a variable (not equality like in math)
- Shorthand operators like `+=`, `-=`, `*=` etc. also exist

**Arithmetic Operators**
- `+` addition, `-` subtraction, `*` multiplication, `/` division (returns float)
- `%` modulus (remainder), `//` floor division (no decimal), `**` exponentiation

**Comparison Operators**
- `==`, `!=`, `>`, `<`, `>=`, `<=` — all return `True` or `False`
- `is` / `is not` — checks if two variables point to the same object
- `in` / `not in` — checks if a value exists within a sequence (string, list, etc.)

**Logical Operators**
- `and` — both conditions must be `True`
- `or` — at least one condition must be `True`
- `not` — negates/flips the boolean value
