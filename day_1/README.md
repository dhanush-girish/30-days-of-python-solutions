Here's a quick summary of Day 2:

**Built-in Functions**
- Python has many globally available built-in functions — no imports needed
- Common ones: `print()`, `len()`, `type()`, `int()`, `float()`, `str()`, `input()`, `min()`, `max()`, `sum()`, `sorted()`, `help()`, `dir()`

**Variables**
- Variables store data in memory; use descriptive, snake_case names (e.g. `first_name`)
- Must start with a letter or `_`, can't start with a number, no special chars or hyphens
- Names are case-sensitive
- You can declare multiple variables in a single line: `a, b, c = 1, 2, 3`
- Use `input()` to get values from the user

**Data Types**
- Use `type()` to check a variable's data type
- Common types: `str`, `int`, `float`, `bool`, `list`, `dict`, `tuple`, `complex`

**Type Casting**
- Convert between types using `int()`, `float()`, `str()`, `list()`, etc.
- e.g. `float(10)` → `10.0`, `str(10)` → `'10'`, `list('hello')` → `['h','e','l','l','o']`

**Number Types**
- **int** — whole numbers (e.g. `-3, 0, 5`)
- **float** — decimals (e.g. `3.14`)
- **complex** — e.g. `1 + 2j`
